Using Gaggle

Using Gaggle can be as simple as pressing the paper plane tab and pressing the "Start Logging"" button at the bottom of the screen. For more information, see the Basic Flight Recording page.

To use Gaggle, you will need to turn GPS on. If GPS is not enabled when you start Gaggle, it will prompt you to switch GPS on.

If you device has a barometer, this will be used to calculate your altitude. Basic altitude calculations are possible via GPS, but these are rought at best and should not be over-relied on.

Your device accelerometer will be used to calculate speed, etc
