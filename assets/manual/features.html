Features now ready for beta-test:
  • Leonardo Live usage
  • Leonardo upload
  • Flight logging
  • View flights on map (live or stored)
  • Send flights via email (various formats)
  • Delete flights
  • Basic flight instruments (not yet reconfigurable)
  • Add/Edit/List/Delete waypoints
  • Auto import waypoints when the browser visits .wpt, .dat & .cup links on web, or kept on local file system.
  • User can choose their prefered lat/lon display format.
  • Waypoints can have a 'type' set - Gaggle will automatically show GR required to reach nearest LZ waypoints
  • Allow the user to view IGC files from the net or local filesystem
  • For testing purposes, you can enable "Simulate GPS Data" in the preferences.  It will playback a prestored flight.
  • "GR to LZ" and "GR to next waypoint" info fields
